## 🌟 What is terraformic?

terraformic helps you set up and control Kubernetes clusters on any cloud provider. It works with popular cloud platforms like AWS, Google Cloud, and Hetzner. With terraformic, you can manage cloud storage, virtual networks, and Kubernetes containers without needing deep programming skills.

It uses Terraform scripts to make deployments easier and more consistent. This tool saves time by automating tasks that usually take hours when done by hand.

## 💡 About Terraform and Kubernetes

terraformic uses two key technologies:

- **Terraform**: A tool to write cloud setup steps as plain text files. terraformic creates these files automatically for you.  
- **Kubernetes**: A system to run and manage containers. Containers hold your apps and services in a safe environment on the cloud.

By combining these, terraformic makes it simple to run powerful cloud setups without learning the technical details.
